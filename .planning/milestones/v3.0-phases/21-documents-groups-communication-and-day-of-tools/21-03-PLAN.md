**src/lib/hooks/useEventDocuments.ts:**
Create hooks using TanStack Query (useQuery/useMutation pattern matching existing hooks like useEventProject):

- `useDocumentRequirements(eventProjectId)` — GET `/api/events/projects/${id}/documents`. Returns { requirements, stats: { totalRequirements, totalRegistrations, completionPercentage } }.
- `useDocumentMatrix(eventProjectId)` — GET `/api/events/projects/${id}/documents/completions`. Returns matrix data. Poll every 30s (refetchInterval: 30000).
- `useToggleCompletion()` — PATCH mutation to `/api/events/projects/${id}/documents/completions`. Optimistic update: immediately toggle the cell, revert on error.
- `useSendReminders(eventProjectId)` — POST mutation to `/api/events/projects/${id}/documents/reminders`.
- `useCreateDocumentRequirement()` — POST mutation to `/api/events/projects/${id}/documents`.
- `useUpdateDocumentRequirement()` — PUT mutation.
- `useDeleteDocumentRequirement()` — DELETE mutation.
- `useComplianceItems(eventProjectId)` — GET `/api/events/projects/${id}/compliance`.
- `useUpsertComplianceItem()` — POST/PUT mutation.
- `useDeleteComplianceItem()` — DELETE mutation.

All mutations invalidate the relevant query cache on success.
