**Create src/lib/hooks/useNotificationRules.ts:**
- `useNotificationRules(eventProjectId: string)` — GET /api/events/projects/{id}/notifications. Returns { rules, isLoading }. queryKey: ['notification-rules', eventProjectId].
- `useNotificationMutations(eventProjectId: string)` — Returns:
  - createRule: POST, invalidates rules query
  - updateRule: PATCH to /notifications/{ruleId}, invalidates
  - deleteRule: DELETE to /notifications/{ruleId}, optimistic removal
  - submitForApproval: POST to /notifications/{ruleId}/approve with { action: 'submit' }
  - approveRule: POST to /notifications/{ruleId}/approve with { action: 'approve' }
  - cancelRule: POST to /notifications/{ruleId}/approve with { action: 'cancel' }
- `useAIDraft(eventProjectId: string)` — Mutation that calls POST /api/events/ai/generate-summary (or a dedicated notification draft endpoint) with event context. Returns { subject, body }.
