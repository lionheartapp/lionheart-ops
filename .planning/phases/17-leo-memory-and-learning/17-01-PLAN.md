<task type="auto">
  <name>Task 2: Create embedding service with pgvector infrastructure</name>
  <files>src/lib/services/ai/embeddingService.ts</files>
  <action>
Create `src/lib/services/ai/embeddingService.ts` with the following:

**Install dependency first:** `npm install pgvector` (provides `toSql` function for vector serialization).

**pgvector setup function** — `ensurePgvector()`:
- Run `CREATE EXTENSION IF NOT EXISTS vector` via rawPrisma.$executeRawUnsafe
- Run HNSW index creation for each embedding column (idempotent with IF NOT EXISTS):
  - `CREATE INDEX IF NOT EXISTS idx_ticket_embedding ON "Ticket" USING hnsw (embedding vector_cosine_ops) WITH (m = 16, ef_construction = 200)`
  - Same for CalendarEvent, InventoryItem, ConversationMessage, UserMemoryFact
- Cache a boolean `pgvectorInitialized` to avoid re-running every call
- Wrap in try/catch — log error but don't crash if extension already exists

**generateEmbedding(text: string): Promise<number[]>**
- Import `GoogleGenAI` from `@google/genai`
- Use apiKey from `process.env.GEMINI_API_KEY || process.env.NEXT_PUBLIC_GEMINI_API_KEY`
- Call `client.models.embedContent({ model: 'text-embedding-004', contents: text, config: { outputDimensionality: 768 } })` (Matryoshka truncation from 3072 to 768)
- Return the embedding values array
- Return empty array if no API key configured (graceful degradation)
- Truncate input text to 2048 chars max before embedding (API limit safety)

**storeEmbedding(tableName: string, recordId: string, embedding: number[]): Promise<void>**
- Use `toSql` from `pgvector` to serialize the array
- Run `UPDATE "${tableName}" SET embedding = $1::vector WHERE id = $2` via rawPrisma.$executeRawUnsafe
- Call ensurePgvector() first (lazy init)

**searchSimilar(tableName: string, queryEmbedding: number[], opts: { limit?: number, orgId?: string, filters?: string }): Promise<Array<{ id: string, similarity: number }>>**
- Call ensurePgvector() first
- Use `toSql` from `pgvector` for the query vector
- Build SQL: `SELECT id, 1 - (embedding <=> $1::vector) AS similarity FROM "${tableName}" WHERE embedding IS NOT NULL`
- Add `AND "organizationId" = $2` if orgId provided
- Add custom filters string if provided
- Add `ORDER BY embedding <=> $1::vector LIMIT $N`
- Default limit: 10
- Return array of { id, similarity } sorted by similarity descending

**generateAndStoreEmbedding(tableName: string, recordId: string, text: string): Promise<void>**
- Convenience function: calls generateEmbedding then storeEmbedding
- Skip silently if text is empty or API key missing

Export all 5 functions.
