   ```prisma
   model TicketComment {
     id             String   @id @default(cuid())
     organizationId String
     ticketId       String
     authorId       String
     body           String
     createdAt      DateTime @default(now())
     updatedAt      DateTime @updatedAt

     ticket  Ticket @relation(fields: [ticketId], references: [id], onDelete: Cascade)
     author  User   @relation("TicketCommentAuthor", fields: [authorId], references: [id], onDelete: Cascade)

     @@index([ticketId, createdAt])
     @@index([organizationId])
   }

   model TicketAttachment {
     id             String   @id @default(cuid())
     organizationId String
     ticketId       String
     uploadedById   String
     fileName       String
     fileUrl        String
     contentType    String
     sizeBytes      Int
     createdAt      DateTime @default(now())

     ticket      Ticket @relation(fields: [ticketId], references: [id], onDelete: Cascade)
     uploadedBy  User   @relation("TicketAttachmentUploader", fields: [uploadedById], references: [id], onDelete: Cascade)

     @@index([ticketId, createdAt])
     @@index([organizationId])
   }
   ```

   Add the corresponding relation arrays to the Ticket model:
   ```prisma
   comments    TicketComment[]
   attachments TicketAttachment[]
   ```


   ```typescript
   include: {
   ```


   ```typescript
   if (validated.search) {
     const s = validated.search.trim()
     if (s) {
       where.AND = [
         ...(where.AND || []),
   ```

2. **Create `src/lib/services/ticketAttachmentService.ts`:**
   - Import `prisma` from `@/lib/db`, `z` from `zod`, `validateFileUpload` from `@/lib/validation/file-upload`, `ALLOWED_DOCUMENT_TYPES` from `@/lib/validation/file-upload`
   - `CreateAttachmentSchema = z.object({ fileName: z.string().min(1).max(255), fileBase64: z.string().min(1), contentType: z.string().min(1) })`
   - `listAttachments(ticketId: string)`: `prisma.ticketAttachment.findMany({ where: { ticketId }, orderBy: { createdAt: 'desc' }, include: { uploadedBy: { select: { id: true, name: true } } } })`
   - `createAttachment(ticketId: string, uploadedById: string, input: { fileName, fileBase64, contentType }, organizationId: string)`:
     a. Validate with CreateAttachmentSchema
     b. Decode base64 to Buffer: `Buffer.from(validated.fileBase64, 'base64')`
     c. Validate file: `validateFileUpload({ size: buffer.length, type: validated.contentType, name: validated.fileName }, { allowedTypes: ALLOWED_DOCUMENT_TYPES, maxSizeBytes: 10 * 1024 * 1024 })` — reject if invalid
     d. Upload to Supabase Storage via the supabase client (import from `@/lib/supabase` or use the same client pattern as storageService.ts). Upload path: `ticket-attachments/${organizationId}/${ticketId}/${Date.now()}-${validated.fileName}`
     e. Get the public URL from the upload result
     f. Create DB record: `prisma.ticketAttachment.create({ data: { ticketId, uploadedById, fileName: validated.fileName, fileUrl: publicUrl, contentType: validated.contentType, sizeBytes: buffer.length, organizationId } })`
     g. Return the attachment record
