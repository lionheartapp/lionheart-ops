Create the Events list page, the EventProject 8-tab workspace page with functional Schedule and Tasks tabs, empty states for future-phase tabs, activity log component, the EventSeriesDrawer for recurring series creation, and the CalendarEvent-to-EventProject deep-link in the calendar view.

Purpose: This is the main UI that staff interact with. Events becomes the top-level navigation concept ("Events Are the Product"), and the 8-tab workspace is where all event management happens. The deep-link closes the EVNT-08 circuit so bridged calendar events navigate back to their EventProject.
Output: Events list page, event detail page with 8 tabs (2 functional, 5 empty-state shells), activity log component, create modal, series drawer, calendar deep-link.
</objective>
