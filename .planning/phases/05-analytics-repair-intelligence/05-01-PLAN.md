2. **Create `src/lib/services/maintenanceAnalyticsService.ts`**:
   All functions accept `(orgId: string, options?: { campusId?: string; schoolId?: string; months?: number })`.
   Use `rawPrisma` (not the org-scoped `prisma`) since these are cross-entity aggregations running outside a route context. Apply `organizationId: orgId` and `deletedAt: null` manually in every where clause.

   **`getTicketsByStatus(orgId, opts)`** — ANALYTICS-01:
   - Query: `rawPrisma.maintenanceTicket.groupBy({ by: ['status', 'schoolId'], where: { organizationId: orgId, deletedAt: null, ...(campusId ? { school: { campusId } } : {}) } })`
   - Join school names via `rawPrisma.school.findMany` separately
   - Return: `{ statuses: string[], campuses: { campusName: string, counts: Record<status, number> }[] }`

   **`getResolutionTimeByCategory(orgId, opts)`** — ANALYTICS-02:
   - Query DONE tickets: `rawPrisma.maintenanceTicket.findMany({ where: { organizationId: orgId, status: 'DONE', deletedAt: null, createdAt: { gte: cutoff } }, select: { category: true, createdAt: true, updatedAt: true } })`
   - Compute resolution hours as `(updatedAt - createdAt) / 3_600_000`
   - Group by category, compute average
   - Return: `{ category: string, avgHours: number }[]` sorted by avgHours desc

   **`getTechnicianWorkload(orgId, opts)`** — ANALYTICS-03:
   - Active tickets per tech: `rawPrisma.maintenanceTicket.groupBy({ by: ['assignedToId'], where: { organizationId: orgId, deletedAt: null, status: { notIn: ['DONE', 'CANCELLED'] }, assignedToId: { not: null } } })`
   - Hours logged this week and month: `rawPrisma.maintenanceLaborEntry.groupBy({ by: ['technicianId'], where: { organizationId: orgId, startTime: { gte: weekStart } }, _sum: { durationMinutes: true } })`
   - Join user names via `rawPrisma.user.findMany`
   - Return: `{ technicianId: string, name: string, activeTickets: number, hoursThisWeek: number, hoursThisMonth: number }[]`

   **`getPmComplianceRate(orgId, opts)`** — ANALYTICS-04:
   - PM tickets: `rawPrisma.maintenanceTicket.findMany({ where: { organizationId: orgId, pmScheduleId: { not: null }, pmScheduledDueDate: { lte: now }, deletedAt: null }, select: { status: true, pmScheduledDueDate: true, updatedAt: true } })`
   - Completed on time: `status === 'DONE' && updatedAt <= pmScheduledDueDate`
   - Overdue: `status !== 'DONE' && pmScheduledDueDate < now` OR `status === 'DONE' && updatedAt > pmScheduledDueDate`
   - Return: `{ completedOnTime: number, overdue: number, pending: number, complianceRate: number }`

   **`getLaborHoursByMonth(orgId, opts)`** — ANALYTICS-05:
   - Query last N months of labor entries with building join via ticket
   - `rawPrisma.maintenanceLaborEntry.findMany({ where: { organizationId: orgId, startTime: { gte: cutoff } }, include: { ticket: { select: { buildingId: true, category: true, building: { select: { name: true } } } } } })`
   - Group by month (format: "2026-01") and building name
   - Return: `{ month: string, buildings: { buildingName: string, hours: number }[] }[]`

   **`getCostByBuilding(orgId, opts)`** — ANALYTICS-06:
   - Labor costs: join `maintenanceLaborEntry` → `technicianProfile.loadedHourlyRate`, group by ticket.buildingId per month
   - Material costs: `maintenanceCostEntry.amount` per ticket.buildingId per month
   - Return: `{ month: string, buildings: { buildingName: string, laborCost: number, materialsCost: number, total: number }[] }[]`

   **`getTopTicketLocations(orgId, opts)`** — ANALYTICS-07:
   - `rawPrisma.maintenanceTicket.groupBy({ by: ['buildingId', 'areaId', 'roomId'], where: { organizationId: orgId, deletedAt: null, createdAt: { gte: cutoff } }, _count: { id: true }, orderBy: { _count: { id: 'desc' } }, take: 10 })`
   - Join building/area/room names separately
   - Return: `{ rank: number, locationLabel: string, ticketCount: number }[]` (top 10)

   **`getCategoryBreakdown(orgId, opts)`** — ANALYTICS-08:
   - `rawPrisma.maintenanceTicket.groupBy({ by: ['category'], where: { organizationId: orgId, deletedAt: null, createdAt: { gte: cutoff } }, _count: { id: true } })`
   - Return: `{ category: string, count: number, pct: number }[]` (pct = count / total * 100)

   Export a combined `getAllAnalytics(orgId, opts)` that calls all 8 functions in parallel via `Promise.all`.
