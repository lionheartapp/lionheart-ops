```javascript
  return { resource: parts[0], action: parts[1], scope: parts[2] || 'global' }
}

```

Permission table compound unique: `@@unique([resource, action, scope])` named `resource_action_scope`
RolePermission compound unique: `@@unique([roleId, permissionId])` named `roleId_permissionId`
