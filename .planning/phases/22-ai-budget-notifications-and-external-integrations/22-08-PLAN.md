**Create src/lib/hooks/useEventTemplates.ts:**
- `useTemplates(opts?: { eventType?: string })` — GET /api/events/templates with optional type filter. queryKey: ['event-templates', eventType]. Returns { templates, isLoading }.
- `useTemplate(templateId: string)` — GET /api/events/templates/{id}. Returns { template, isLoading }. Enabled only when templateId is provided.
- `useTemplateMutations()` — Returns:
  - saveAsTemplate: POST to /api/events/templates with { eventProjectId, name, description?, eventType? }. Invalidates templates query.
  - createFromTemplate: POST to /api/events/templates/{id} with { title, startsAt, endsAt, locationText? }. Returns new eventProjectId.
  - deleteTemplate: DELETE to /api/events/templates/{id}. Optimistic removal.
